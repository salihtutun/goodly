API Reference

83 endpoints across 3 route modules — all rate-limited with feature flags

Authentication

POST /api/auth/login Public

Authenticate with email + password. Returns JWT in HttpOnly cookie + user object.

Rate: 10/minBody: email, password
POST /api/auth/register Public

Create new account. Optional website URL triggers first audit. Sends verification email.

Rate: 5/minBody: name, email, password, url?
GET /api/auth/me Auth Required

Get current user profile from JWT cookie.

Rate: 30/min
POST /api/auth/logout Auth Required

Clear JWT cookie.

POST /api/auth/forgot-password Public

Send password reset email. Always returns success (prevents email enumeration).

POST /api/auth/reset-password Public

Reset password with token from email.

GET /api/auth/verify-email Public

Verify email with token. Redirects to frontend login.

POST /api/auth/onboarded Auth Required

Mark user as onboarded.

Audits

POST /api/audits Auth Required

Run a new SEO audit on a URL. Returns 50+ signals, score, issues, revenue impact.

GET /api/audits Auth Required

List user's audit history.

GET /api/audits/:id Auth Required

Get audit detail with AI recommendations.

POST /api/audits/public Public

Run a public audit (no login required). Triggers nurture email sequence.

POST /api/audits/:id/share Auth Required

Generate shareable link for an audit.

GET /api/shared/:token Public

View a shared audit by token.

GET /api/audits/:id/pdf Auth Required

Download audit as PDF report.
